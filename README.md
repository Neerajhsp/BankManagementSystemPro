# 🏦 Bank Management System Pro

A professional console-based Banking Management System developed in **C++** using **Object-Oriented Programming (OOP)** concepts and **CMake**.

## 🚀 Features

- Admin Login
- Create Account
- Savings & Current Account
- Deposit Money
- Withdraw Money
- Transfer Money
- Balance Inquiry
- Mini Statement
- Transaction History
- Customer Search
- Update Customer Details
- Delete Account
- Interest Calculator
- Loan Eligibility Calculator
- Monthly Report
- Export Reports
- File Handling for Data Storage

## 🛠 Technologies Used

- C++
- OOP
- STL
- File Handling
- CMake
- VS Code

## 📁 Project Structure

```
BankManagementSystemPro
│
├── include/
├── src/
├── CMakeLists.txt
└── .gitignore
```

## ▶️ Build & Run

```bash
mkdir build
cd build
cmake ..
cmake --build .
./BankManagementSystemPro
```
